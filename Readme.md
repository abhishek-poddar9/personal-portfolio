## Abhishek Poddar - Personal Portfolio

A modern, responsive personal portfolio website built to showcase my projects, technical skills, and certifications with a strong focus on clean UI, smooth navigation, and performance optimization.


## Features

- Responsive Design (Mobile-first)
- Clean and modern UI
- Smooth navigation and scrolling
- Project showcase with GitHub links
- Resume view and download
- Optimized performance

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS
- Vite
- Git & GitHub
